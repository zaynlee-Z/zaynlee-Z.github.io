查看三方SDK

Created2024-09-13|Updated2024-09-13
|Post Views:

  1. 获取IPA
    1.1 Apple Configuration 从MAC的Apple Store下载，利用App下载机制导出ipa包

    1.2 iMazing/爱思助手/iTools 三方软件下载IPA，会被三方重签名，软件更新及时性&内容可能会与官方下载的不一致(Apple Store已下架的应用也可以尝试在此下载)

  2. 解压缩
    将下载好的包直接解压后会获得APP文件

  3. 查看当前APP包含的SDK

3.1 打开文件com.luckygames.superbrain-2\Payload\SuperBrain.app\Data\ScriptingAssemblies.json(包名已实际解压后的为准，ScriptingAssemblies.json为当前包内所有的SDK)

3.2 处理json文件
3.2.1 取names的值，过滤unity相关SDK

3.2.2 unity常见依赖库，也可以过滤


3.2.3 过滤后的为三方SDK
3.2.4 注意：fairyGUI 是一个用于 Unity、Cocos、Egret、Laya 等引擎的 GUI 编辑器和 UI 框架，fairyGUI及子文件不属于SDK范畴
